# BorderlineTAF v0.2

Kleiner Tasker-App-Factory-Prototyp als lokale HTML-Panel-Schicht für Android. Kein natives Android-Projekt, kein Launcher, kein IME, keine Accessibility-Abhängigkeit.

## Quickstart
1. Alle Dateien aus diesem Repo-Bereich auf das Handy kopieren.
2. In Tasker eine **Scene** mit einem **WebView** anlegen.
3. `borderline_taf/index.html` lokal in die WebView laden.
4. Einen Start-Task bauen (Scene anzeigen, optional Variablen setzen).
5. Mit **Tasker App Factory** als APK exportieren.
